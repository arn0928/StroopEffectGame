# 終端色彩判斷遊戲 (Stroop Effect Game)

這是一款基於 Java 開發的終端機文字遊戲。靈感來自於心理學中的「斯特魯普效應 (Stroop Effect)」，考驗玩家的反應速度與專注力！

## 🌟 遊戲特色

- **四大模式**：
  - `普通模式`：100 個主線關卡，挑戰難度逐漸升級。
  - `無盡模式`：難度全開的無限挑戰！通關 100 關後解鎖，挑戰你的極限，爭取最高金幣！
  - `多人模式`：與好友在不同終端機上輸入相同的「種子碼」，就能產生一模一樣的關卡序列進行速度對決！
  - `練習模式`：無限制時間與血量，供玩家熟悉題目與找回手感。
- **精準數據統計**：遊戲會幫你紀錄各種模式的「總正確率」、「各顏色平均作答時間」以及「反轉邏輯的反應時間」，隨時檢視自己的能力成長！
- **商店與道具**：破關獲得金幣，可於商店購買「裝甲包」、「全對」、「水影片」、「白飯吃到飽（無盡保險）」以及神秘的「封印之書」。

## 🎮 遊戲規則與操作

螢幕上會隨機出現帶有顏色的中文字，直接按下對應數字鍵即可作答，不需再按 Enter。
- **若「文字字義」等於「顯示顏色」**：按下 `1` 判斷為真。
- **若「文字字義」不等於「顯示顏色」**：按下 `2` 判斷為假。

### 金幣規則
- **普通模式**：答對 +1 金幣（特定關卡 +10 金幣）
- **無盡模式**：答對 +5 金幣 / **答錯 -5 金幣**（可購買白飯吃到飽避免扣錢）
- **多人 / 練習模式**：無金幣獎勵（且不會消耗持有道具）

### 快捷鍵操作：
- `3` ：暫停遊戲（注意：多人模式下暫停會導致後續題目不同步）
- `4` ：儲存進度並退出本次遊玩
- `5` ：使用「全對」道具（無限制模式下無法使用）
- `6` ：啟動「封印之書(禁術)」（需先購買）
