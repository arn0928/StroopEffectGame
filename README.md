# 終端色彩判斷遊戲 (Stroop Effect Game)

這是一款基於 Java 開發的終端機文字遊戲。靈感來自於心理學中的「斯特魯普效應 (Stroop Effect)」，玩家必須在緊迫的時間內，快速判斷螢幕上顯示的「文字字義」與「文字顏色」是否相符，考驗玩家的反應速度與專注力！

## 🌟 遊戲特色

- **豐富的關卡與難度遞增**：共 100 個主線關卡，隨著關卡推進，將面臨新顏色（粉色）、邏輯反轉（反轉術式）、視覺干擾（背景色）以及縮短的時間獎勵等挑戰。
- **無盡模式**：通關 100 關後解鎖，挑戰你的極限，爭取最高金幣！
- **商店與道具系統**：破關獲得金幣，可於商店購買「裝甲包（血量）」、「全對（免死金牌）」、「水影片（加時）」以及神秘的「封印之書（時間暫停）」。
- **自動存檔點機制**：每 20 關解鎖一個存檔點，中途離開也不怕心血白費。防作弊系統確保玩家無法透過暫停遊戲來偷看題目。

## 🎮 遊戲規則與操作

螢幕上會隨機出現帶有顏色的中文字。
- **若「文字字義」等於「顯示顏色」**（例如：紅色的「紅」字）：直接按下 `Enter` 判斷為真。
- **若「文字字義」不等於「顯示顏色」**（例如：藍色的「紅」字）：輸入 `'` (單引號) 後按下 `Enter` 判斷為假。

### 快捷鍵操作：
- `[` ：暫停遊戲（解除暫停時會重新生成題目以防作弊）
- `]` ：儲存進度並返回主選單
- `\` ：使用「全對」道具（需先於商店購買並裝備）
